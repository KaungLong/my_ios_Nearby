# NearBy
---
A simple app  practice that uses an MKMapView to search for nearby places that match a keyword.

## Environment
---
Xcode 13.3 +
## Features structure
---
- UIKit programmatically
- MKMapView
- Core Location
- Set app access permissions by setting Info.plist
